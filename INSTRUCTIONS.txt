SIC PROGRAM TO MANUALLY INPUT A NUMBER(IN MEMORY) AND PERFORM THE FOLLOWING OPERATIONS
1. LOAD IT INTO ACCUMULATOR
2. MULTIPLY IT BY 4
3. SUBTRACT 9(BASE 10) FROM IT.
4. STORE THE RESULT IN MEMORY

INPUT MEMORY LOCATION - BETA
OUTPUT MEMORY LOCATION - ALPHA

CODE:	A.ASM


PROG2 START 00000
      LDA BETA
      MUL #4
      SUB #9
      STA ALPHA
ALPHA RESW 1
BETA RESW 1
